-> 区块链是链式数据结构的去中心化的分布式共享账本系统，每个节点计算机都持有全球的比特币交易账本，每笔交易都需要全部节点授权通过后才能完成。区块链是比特币的底层技术，比特币是世界上第一条区块链，也是第一代区块链，以太坊是第二代区块链(智能合约和DApp——分布式应用程序)
-> 区块链的本质是从专人记账转向共同记账，民主交易
去中心化 -> p2p -> 分布式计算 -> 区块链 -> 比特币
|区块 -> 写满交易记录，以链式数据结构按时间戳顺序相连，形成区块链。每个区块都是在前一个区块的数据基础上生成的，该机制保证了区块链数据的唯一性。
    -> 每个矿工都必须等前一个区块生成之后，才能根据新生成区块的数据开始计算符合条件(哈希算法)的随机数(挖矿), 保证了挖矿的公平性
-> 非对称加密机制保证了私钥的安全性

|区块链核心技术
    1.p2p -> 去中心化
    2.分布式存储
    3.加密算法 -> 非对称加密，Merkle树，时间戳，哈希算法
    4.共识算法 -> PoW(工作量证明机制)
    5.智能合约 -> 通过自动化脚本操作数据

-> 区块链的未来是自组织(DAO)
