|贝叶斯方法 -> 概率是一个人对某件事的信念强度，概率是'主观的'。贝叶斯方法容易得出局部最优解
    -> 频率学派认为概率是客观且固定的，进行一定数量的重复实验后，如果出现某个现象的次数与总次数的比趋于某个值(如抛硬币)，那么这个比值就会倾向于固定(上帝视角)
    -> 贝叶斯方法认为频率学派的观点是不对的，计算概率需要加入'先验概率'的考虑(世俗视角)
    |先验概率 -> 先设定一个假设，然后通过实验证明或推翻这个假设，这就是后验。随后旧的后验会成为一个新的先验，如此重复，就得到了贝叶斯公式
        P(A|B) = P(B|A)*P(A)/P(B) #P(A)为先验概率,P(A|B)为后验概率
        |条件概率 -> 在事件B已发生的前提条件下，事件A发生的概率。表达式为P(A|B)
        -> '知识'就是先验概率. 在缺乏充分信息的情况下，贝叶斯方法是最优的推理方法
        |生活范例 -> 炮兵发炮时，先根据计算和经验把炮管调整到一个可能命中的角度(先验概率)，然后再根据炮弹的实际落点进行调整(后验概率)，经过几次射击和调整后就能命中目标了
        |应用场景
            1.自然语言处理中的写作用词风格识别(词频)
            2.自然语言处理中的二义性语义判断
            3.自然语言处理中的语音识别(贝叶斯方法 + 马尔科夫链)
                |马尔科夫性质 -> 当一个随机过程在给定现在状态和所有过去状态的情况下，其未来状态的条件概率分布仅由当前状态决定(无记忆性)
                |马尔科夫过程 -> 拥有马尔科夫性质的过程，如马尔科夫链，布朗运动
                |马尔科夫链 -> 直观的随机过程建模方法。句子中的每个字符都只由前一个字符决定
                    |转移 -> 状态的变化。系统整体模型由一个'状态转移矩阵'确定，此矩阵是马尔科夫链的'核心'
                    -> 马尔科夫链由一系列满足马尔科夫性质的转移组成，这些转移符合某种概率分布
                    -> 马尔科夫链是入门概率建模和数据科学的良好开端
            4.海难搜救(将可能区域划分为格子，A为目标在某格子里的概率，B为目标被搜索到的概率，总的搜救成功的概率为1。一个格子搜索后没有成功，本身概率降低，其他格子区域搜索成功的概率提升，不断洗牌)
            5.人类大脑很可能就是贝叶斯网络('仅从极少案例就能形成概念')，AI则需要大数据学习

|蒙特卡洛方法 -> 利用'符合规则的随机数'来解决计算问题，与其相对的是确定性算法
    |应用场景 -> 金融工程学，宏观经济学，计算物理学等
    |基本思想 -> 当所求解问题为某随机事件出现的概率时，通过实验方法(从已知概率分布中抽样)，以这种'事件出现的频率'估计其出现的概率
    -> 人为构建的随机性算法，模拟上帝行为
