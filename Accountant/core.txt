\会计
    |概念 -> 关注企业资金流的管理层职位
    |职能 -> 核算和监督
    |目标 -> 财务报表

    |基本假设
        1.会计主体 -> 会计工作服务的特定对象，是企业会计确认，计量和报告的空间范围(企业，资金)
            -> 法律主体(企业)一定是会计主体; 会计主体不一定是法律主体
        2.持续经营 -> 在可预见的将来，企业将会按当前的规模和状态持续经营下去，不会停业和大规模削减业务。这个假设意味着会计主体将按照既定用途使用资产，按既定的合约条件清偿债务
            -> 持续经营是会计分期的前提
        3.会计分期 -> 将企业的持续经营活动平均划分为一个个连续的时间间隔。目的是分期结算账目和编制财务报告，从而及时向财务报告使用者提供企业财务状况，经营成果和现金流信息
            -> 分为年度(公历1月1日到12月31日)和中期(短于年度的分期: 月度，季度，半年度)
        4.货币计量 -> 会计主体在确认，计量和报告时以货币计量，反应会计主体的生产经营活动

    |会计基础 -> 会计分期假设意义重大，正因为有了会计分期，才产生了本期和非本期的区别，继而产生了权责发生制和收付实现制，然后出现了应收，应付，预收，预付，折旧和摊销等会计处理方法
        1.权责发生制 -> 以实际发生(收钱的权力和付钱的责任发生了)而非以实际收支作为确认收入和费用的标准。适用于企业会计和行政事业单位中的财务会计
        2.收付实现制 -> 以实际收到或支付现金作为确认收入和费用的标准。适用于行政事业单位中的预算会计

    |会计要素 -> 财务状况: 资产，负债和所有者权益; 经营成果: 收入，费用和利润
        1.资产 -> 由企业已完成的交易(双方的)或事项(单方的)形成的企业拥有或控制(租赁)的资源，资产'预期(不一定)'会给企业带来经济利益
            |资产的确认条件
                1.与该资源有关的经济利益很可能(大于50%的几率)流入企业
                2.该资源的成本或价值能够可靠地计量(价值确定性)
            |分类
                1.流动性资产 -> 预计能在一个正常营业周期(一般指一年)内变现，出售或耗用的资产，或者主要为交易目的而持有的资产(股票，基金)，或者在资产负债表日(每年1月1日)起一年之内能变现的资产，以及自资产负债表日起一年内交换其他资产或清偿负债的能力不受限制的现金或现金等价物(股票，黄金债券)
                2.非流动性资产 -> 流动性资产以外的资产(房产)
        2.负债 -> 由企业已完成的交易或事项形成的企业现时(已在法理上形成的)义务，'预期(不一定)'会导致企业经济利益流出
            |负债的确认条件
                1.与该义务有关的经济利益很可能(大于50%的几率)流出企业
                2.未来流出的经济利益的金额能够可靠地计量
            |负债的分类
                1.流动性负债 -> 预计在一个正常营业周期(一般指一年)内偿还，或者主要为交易目的而持有，或者自资产负债表日起一年内到期应予以清偿，或者企业无权自主地将清偿推迟至资产负债表日以后一年以上的负债
                2.非流动性负债 -> 流动性负债以外的负债
        3.所有者权益 -> 等于'企业资产减去负债(债权人权益)'，也就是'净资产'。反映'股东'在企业资产中享有的经济利益
            |特征
                1.除非发生减资(股东撤资)，清算(企业破产)或分派现金股利(分红)，企业不需要偿还所有者权益
                2.企业清算时，只有在偿还企业负债后，才会将所有者权益返还所有者
                3.所有者凭借所有者权益能参与企业利润非配(分红)，'债权人没有分红权力'
            |所有者权益确认条件 -> 取决于资产，负债，收入，费用等要素的确认和计量
            |所有者权益分类
                1.所有者投入的资本
                2.直接计入所有者权益的利得和损失，留存利益(盈余公积，未分配利润)等
                    |盈余公积 -> Surplus Reserve. 包括法定盈余公积，任意盈余公积和法定公益金
                        1.法定盈余公积 -> 上市公司的法定盈余公积按照税后利润的10%提取，法定盈余公积累计额已达注册资本的50%时可以不再提取
                        2.任意盈余公积 -> 任意盈余公积主要是上市公司按照股东大会的决议提取。法定盈余公积和任意盈余公积的区别就在于其各自计提的依据不同。前者以国家的法律或行政规章为依据提取，后者则由公司自行决定提取
                        3.法定公益金 -> 用于职工福利支出。'2006年公司法修改后，法定公益金已经取消'，改为'任意公积金'的内容
                        |用途
                            1.弥补亏损
                            2.转增资本
                            3.扩大生效
                            -> 盈余公积的提取实际上是企业当期实现的净利润向投资者分配利润的一种限制。提取盈余公积本身就属于利润分配的一部分，一经提取形成盈余公积后，在一般情况下不得用于向投资者分配利润或股利。当有特殊情况(企业当年没有利润)需要用盈余公积分配股利，必须符合下列条件： (1)用盈余公积弥补亏损后，该项公积金仍有结余。 (2)用盈余公积分配股利时，股利率不能太高，不得超过股票面值的6%。 (3)分配股利后，法定盈余公积金不得低于注册资本的25%
                            -> 企业提取的盈余公积，无论是用于弥补亏损，还是用于转增资本，只不过是在企业所有者权益内部结构的转换，如企业以盈余公积弥补亏损时，实际是减少盈余公积留存的数额，以此抵补未弥补亏损的数额，并不引起企业所有者权益总额的变动；企业以盈余公积转增资本时，也只是减少盈余公积结存的数额，但同时增加企业实收资本或股本的数额，也'并不引起所有者权益总额的变动'
        4.收入
        5.费用 -> 成本
        6.利润 -> 等于'收入减去费用'
